# 🌟 Project Overview / Descripción General del Proyecto
El proyecto es un sistema integral diseñado para automatizar y gestionar diversas tareas y operaciones, incluyendo procesos de alta y baja, manejo de solicitudes e interacciones con la base de datos. El sistema consta de múltiples scripts de PowerShell, scripts en PHP y archivos SQL, cada uno con un propósito específico dentro de la funcionalidad global del proyecto. El objetivo principal es proporcionar una forma eficiente y optimizada de gestionar recursos, usuarios y datos, facilitando la ejecución de tareas y operaciones diarias.

## 🚀 Key Features / Funcionalidades Clave
- **Procesamiento de Altas y Bajas**: El sistema incluye scripts de PowerShell (`procesar_altas.ps1` y `procesar_bajas.ps1`) diseñados para automatizar la incorporación y eliminación de usuarios, recursos o registros dentro del ecosistema del proyecto.
- **Gestión de Solicitudes**: El script `insertar_solicitud.php` permite la inserción y el procesamiento de solicitudes, ya sean solicitudes de servicio, de usuario o cualquier tipo de envío de datos.
- **Funciones Reutilizables**: El script `funciones.ps1` proporciona un conjunto de funciones reutilizables que pueden emplearse en todo el proyecto, fomentando la reutilización de código y simplificando el desarrollo y el mantenimiento.
- **Interacciones con la Base de Datos**: El sistema interactúa con bases de datos para almacenar, recuperar y actualizar información, utilizando scripts como `ConsultaTablas.sql` para realizar consultas a las tablas.
- **Gestión de Configuración**: Aunque el archivo `config.ps1` no es accesible, su propósito es proporcionar un punto centralizado para la configuración y posiblemente ejecutar tareas de inicialización o preparación del entorno.
## 🛠️ Tech Stack / Tecnologías Utilizadas
- **PowerShell**: Utilizado para tareas de scripting, incluyendo el procesamiento de altas y bajas, además de proporcionar funciones reutilizables.
- **PHP**: Empleado para manejar solicitudes y potencialmente interactuar con bases de datos.
- **SQL**: Usado para interacciones con la base de datos, incluyendo consultas y modificaciones de datos.
- **Sistemas de Gestión de Bases de Datos**: Como MySQL, PostgreSQL o SQL Server, para almacenamiento y gestión de datos.
- **Active Directory**: Posiblemente utilizado para la gestión de usuarios y autenticación.
- **Módulos de PowerShell**: Diversos módulos para acceso a bases de datos, interacciones con Active Directory y otras funcionalidades.


## 💻 Uso
- **Procesamiento de Altas y Bajas**:Ejecuta `procesar_altas.ps1` y `procesar_bajas.ps1` según sea necesario para gestionar incorporaciones y eliminaciones.
- **Gestión de Solicitudes**: Usa `insertar_solicitud.php` para procesar solicitudes.
- **Interacciones con la Base de Datos**: Ejecuta scripts SQL como `ConsultaTablas.sql` para consultar o modificar datos.
